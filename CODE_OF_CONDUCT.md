# Code of Conduct

## Our Pledge

We are committed to making participation in this project a harassment-free experience for everyone.

## Our Standards

Examples of behavior that contributes to a positive environment:
- Being respectful and constructive.
- Accepting and giving feedback in good faith.
- Focusing on project outcomes, not people.

Examples of unacceptable behavior:
- Harassment, insults, or personal attacks.
- Trolling, intimidation, or discriminatory language.
- Publishing private information without permission.

## Enforcement Responsibilities

Project maintainers are responsible for clarifying and enforcing these standards.

## Scope

This Code of Conduct applies within project spaces and in public spaces when representing the project.

## Reporting

Report behavior issues by opening a private security report or contacting maintainers directly.

## Enforcement

Maintainers may remove, edit, or reject comments, commits, code, issues, and other contributions that violate this Code of Conduct.
